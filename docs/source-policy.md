# Pulse Source Policy

Pulse is a nursing-study organizer. It should help students connect course material to study cards, notes, assignments, and NCLEX-style thinking without becoming a textbook, lecture-slide, or answer-key repository.

## Default Position

- Canadian-first: use fair dealing language, not U.S. fair use language.
- Study support only: Pulse is not medical advice.
- Source pointers over source files.
- Student-authored summaries over copied source text.
- Course-restricted material stays tied to the cohort or course.

## What Pulse Can Store

- Course, module, lecture, chapter, page, and slide references.
- Student-authored study summaries.
- Claim-level source chips.
- Course-specific notes, including professor test preferences.
- Conflict notes when textbook, lecture, and professor guidance differ.

## What Pulse Should Not Share

- Textbook PDFs.
- Full PowerPoint decks.
- Copied textbook chapters.
- Copied slides.
- Answer keys.
- Anything that functions as a replacement for the assigned source.

## Source Hierarchy

1. Professor, course instruction, and lecture slides.
2. Assigned textbook.
3. Trusted outside references.
4. Student notes, when clearly labeled and tied to a source or class date.

## Upload Rule

If source upload exists later, uploaded copyrighted material should be treated as a private processing input. Pulse can keep source anchors and student-authored summaries, but should discard the shared source file and never make it visible to the cohort.

## Conflict Rule

Do not erase source conflicts. Show them clearly.

Example:

- Textbook says HCO3 is 20-30.
- Professor says use 21-28 for this course.
- Study rule: use the professor/course value on course exams.

## Review Rule

Official Pulse card changes require an admin or trusted reviewer. Student submissions can be visible as student-made, but unsourced or disputed claims need a warning label until reviewed.

## Before Wider Launch

Before sharing Pulse beyond a small cohort, get Canadian legal review for the source policy, contributor terms, copyright report flow, and medical-study disclaimer.
